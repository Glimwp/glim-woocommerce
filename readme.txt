A simple WordPress plugin designed to extend the functionality of your theme and provide seamless integration between the theme and the WooCommerce plugin. It enhances the capabilities of your theme by providing additional features and options for contact forms created using WooCommerce.

== Description ==

### GLIM: WooCommerce: the #1 GlimFSE Framework + WooCommerce plugin
A simple extension for [GlimFSE Framework](https://wordpress.org/themes/glimfse/) used as a bridge between WooCommerce and our theme.

== Installation ==
Starting with GLIM: WooCommerce consists of just two steps: installing and setting up the plugin.

### INSTALL GLIM: WooCommerce FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select ‘Add New’;
1. Search for ‘GLIM: WooCommerce’;
1. Activate GLIM: WooCommerce from your Plugins page;
1. Go to ‘after activation’ below.

### INSTALL GLIM: WooCommerce MANUALLY

1. Upload the ‘glim-contact-form-7’ folder to the /wp-content/plugins/ directory;
1. Activate the GLIM: WooCommerce plugin through the ‘Plugins’ menu in WordPress;
1. Go to ‘after activation’ below.

### AFTER ACTIVATION

1. You should see (a notice to start) about GLIM: WooCommerce activation;
1. Go through Appearance -> Theme Options -> Extensions -> WooCommerce and set up the plugin for your site;
1. You’re done!

== Frequently Asked Questions ==

= Why this plugin can't be activated? =

You are probably not running [GlimFSE Framework](https://wordpress.org/themes/glimfse/) or a Child Theme of it.

= Where are the plugin options located? =

The options are located under Extensions panel in Appearance -> Theme Options -> Extensions.
